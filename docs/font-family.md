# Font chữ

| Property      | Value                                   |
| ------------- | --------------------------------------- |
| `font-family` | Một `string`là tên font chữ             |
| `font-size`   | `apx`: cỡ chữ                           |
| `font-style`  | `italic`: Kiểu chữ nghiêng              |
| `font-weight` | `100`,`200`, ...,`900`: Độ đậm tăng dần |
