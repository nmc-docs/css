# Opacity

- **Opacity** là thuộc tính tạo độ mờ cho phần tử

| Property  | Value                                                                       |
| --------- | --------------------------------------------------------------------------- |
| `opacity` | `a`: Với a là giá trị từ 0 đến 1. Độ rõ nét của phần tử tăng dần từ 0 đến 1 |
